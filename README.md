# sso



## Installing

```shell
$ composer require duc_cnzj/sso -vvv
```

## Configuration

| 环境变量         | 作用                                | 默认值      |
| ---------------- | ----------------------------------- | ----------- |
| SSO_PROJECT_NAME | 项目名称                            |             |
| SSO_BASE_URL     | sso domain地址                      |                      |
| TOKEN_FIELD      | 请求的header字段或者session中的字段 | X-Stream-ID |
| SSO_MODE         | api/web                             | Web         |
